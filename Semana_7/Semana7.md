# Alinear un texto a la Izquierda
Ejemplo:
>>> cadena = "bienvenido a mi aplicación".capitalize() 
>>> print cadena.ljust(50, "=")
Bienvenido a mi aplicación UwU=======================
# Alinear un texto a la derecha:
>>> cadena = "bienvenido a mi aplicación".capitalize() 
>>> print cadena.rjust(50, "=")
